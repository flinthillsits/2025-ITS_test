Transit Vehicle On-Board Paratransit Operations Functional Object

Description

'Transit Vehicle On–board Paratransit Operations' forwards paratransit and flexible–route dispatch requests to the operator and forwards acknowledgements to the center. It coordinates with, and assists the operator in managing multi–stop runs associated with demand responsive transit services including paratransit. It collects transit vehicle passenger data and makes it available to the center.

Included In

FHATA On–Board Security Cameras
