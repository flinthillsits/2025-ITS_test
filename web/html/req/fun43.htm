Roadway Mixed Use Crossing Safety Functional Object

Description

'Roadway Mixed Use Crossing Safety' is an advanced infrastructure application that detects pedestrians, cyclists, and other non–motorized users and provides active safety warnings to drivers when cross walks or other intersecting mixed use paths are occupied.

Included In

Manhattan Traffic Signals
Manhattan Bicycle Detection System
