TIC Traveler Telephone Information Functional Object

Description

'TIC Traveler Telephone Information' services voice–based traveler requests for information that supports traveler telephone information systems like 511. It takes requests for traveler information, which could be voice–formatted traveler requests, dual–tone multi–frequency (DTMF)–based requests, or a simple traveler information request, and returns the requested traveler information in the proper format. In addition to servicing requests for traveler information, it also collects and forwards alerts and advisories to traveler telephone information systems.

Included In

KDOT KanDrive Website and 511 Phone Traveler Information System
