Interface: Manhattan Traffic Operations Facility - Manhattan Bicycle Detection System

Information Flow Definitions

mixed use safety warning control  (Planned)  Selected Comm Solution

Configuration and control of equipment that monitors and manages mixed use crossings and provides visual displays and warnings to drivers when non–motorized users are occupying a cross walk or other mixed use path crossing.

mixed use safety warning status  (Planned)  Selected Comm Solution

Current operational status and state of pedestrian crossings and other mixed use path crossing warning systems.

right-of-way request notification  (Planned)  Selected Comm Solution

Notice that a request has occurred for signal prioritization, signal preemption, pedestrian call, multi–modal crossing activation, or other source for right–of–way.

roadway warning system control  (Planned)  Selected Comm Solution

Information used to configure and control roadway warning systems.

roadway warning system status  (Planned)  Selected Comm Solution

Current operating status of roadway warning systems.

traffic detector control  (Planned)  Selected Comm Solution

Information used to configure and control traffic detector systems such as inductive loop detectors and machine vision sensors.

traffic detector data  (Planned)  Selected Comm Solution

Raw and/or processed traffic detector data which allows derivation of traffic flow variables (e.g., speed, volume, and density measures) and associated information (e.g., congestion, potential incidents). This flow includes the traffic data and the operational status of the traffic detectors
