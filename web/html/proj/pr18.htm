K-18 ITS Project

Description

This project will install various ITS components along K–18 from the K–18/K–113 Interchange to the K–18/I–70 Interchange. Project components include installation of speed and count stations, permanent dynamic message signs, and fiber optic cable. In addition, a fiber optic cable will need to be installed from K–18/K–113 Interchange to the Manhattan Traffic Operations Facility.

Status: Identified as Need

Timeframe: Medium Term to Long Term
