Junction City Traffic Signal Interconnects and Upgrades

Description

Junction City currently has one corridor that has signals interconnected; however, the communications has been broken. The initial focus of this project will be repair, replacement and/or upgrade of the existing interconnects to enable corridor–based signal coordination. Additional signal interconnects desired will be identified. This project will also include upgrades of controllers, hardware, etc. to improve signal connectivity and operational efficiency.

Status: In Progress

Timeframe: Short to Medium Term
